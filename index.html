2026 주요활동

공정 엔지니어링 기반 프로젝트 포트폴리오

공정 설계와 운전 조건을 이해하기 위해 시뮬레이션, 센서 데이터, 열유동 해석, 촉매 특성 분석을 연결해 프로젝트를 수행했습니다. 단순 결과 정리보다 변수 설정 → 데이터 확인 → 원인 해석 → 개선 방향 도출의 흐름을 중점적으로 보았습니다.

Project 01

SAF DWSIM 공정모사

Alcohol-to-Jet 기반 SAF 생산 공정을 DWSIM으로 구성하고, H₂ 유량, 반응 조건, 체류시간 변화가 전환율과 생산량에 미치는 영향을 검토했습니다.

  • Feed input, reaction, separation, energy flow 연결
  • 목표 전환율 재현을 위한 운전 변수 조정
  • 지속가능 연료 생산 공정의 mass/energy balance 이해
Project 02

생물학 반응기 디지털 트윈

소형 반응기를 설계·제작하고 Arduino 기반 온도 및 가스 데이터를 수집하여 미생물 분해 시스템의 모니터링과 제어 가능성을 탐구했습니다.

  • 온도, CO₂, CH₄ 센서 데이터 수집
  • 교반 조건과 온도 변화에 따른 반응기 거동 관찰
  • 실험 시스템과 모델 기반 예측의 연결 구조 검토
Project 03

PIML 기반 TIM 열전도 예측

TIM의 열전도 특성을 예측하기 위해 SimScale와 Onshape를 활용해 열전달 해석 조건을 구성하고 온도 응답을 시각화했습니다.

  • 열전달 경계조건 및 입력 변수 정의
  • 온도 분포 기반 설계 개선 방향 도출
  • 물리 기반 모델과 데이터 기반 예측의 결합 가능성 검토
Project 04

CO₂ 수소화 촉매 연구

CO₂ 수소화 촉매의 구조·표면 특성을 전환율, 선택도, 생성물 특성과 연결하여 CCUS 기반 화학 전환 가능성을 해석했습니다.

  • TEM, XRD, TPR, TPD 분석 결과 해석
  • DME/메탄올 생성 특성과 촉매 구조의 관계 검토
  • 활성점, 산점, 금속 분산도와 반응 경향 연결
