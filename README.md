# SKKU SJ E-Portfolio

GitHub Pages용 정적 포트폴리오 사이트입니다.

## 파일 구성

- `index.html`: 페이지 내용
- `styles.css`: 디자인
- `script.js`: 메뉴, 스크롤 애니메이션, 스토리 클릭 기능

## GitHub Pages 배포 방법

1. GitHub에서 새 repository를 만듭니다.
2. 이 ZIP을 풀고, `index.html`, `styles.css`, `script.js`, `README.md`를 repository 최상단에 업로드합니다.
3. Repository → Settings → Pages로 이동합니다.
4. Source를 `Deploy from a branch`로 선택합니다.
5. Branch를 `main`, folder를 `/root`로 선택합니다.
6. Save를 누르면 GitHub Pages 주소가 생성됩니다.

## 수정 방법

- 이름/문구/프로젝트 내용은 `index.html`에서 수정합니다.
- 색상/폰트/카드 디자인은 `styles.css`에서 수정합니다.
- 스토리 원 클릭 내용은 `script.js`의 `storyData`에서 수정합니다.
